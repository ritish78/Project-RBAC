Current Implementation:
1. There are four roles that a user can have for a project:
```
1. SUPER_ADMIN
2. ADMIN
3. MODERATOR
4. VIEWER
```
2. There are actions which only SUPER_ADMIN can do. Then below it are ADMIN, MODERATOR and VIEWER.
